# TryHackMe Rooms

This directory contains technical write-ups and study notes from authorized TryHackMe laboratories.

Each room is organized into its own directory. Individual tasks or privilege-escalation techniques are documented separately to keep the material structured and easy to navigate.

The write-ups focus on:

- Vulnerability identification
- Enumeration methodology
- Exploitation flow
- Operating-system behavior
- Security impact
- Detection opportunities
- Remediation guidance

No flags, challenge answers, active credentials, or malicious payload files are included.

## Ethical Use

All activity documented in this repository was performed inside authorized and intentionally vulnerable training environments.

Target addresses, VPN addresses, credentials, flags, and sensitive values have been removed or replaced with placeholders.
